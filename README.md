# Перечень используемых инструментов с обоснованием выбора

## Инструмент 1: Java
Автоматизированное тестирование удобно осуществлять с помощью этого языка

## Инструмент 2: Selenide или Selenium
Поиск объектов на странице удобно осуществлять с помощью этих инструментов

## Инструмент 3: Junit
Необходимый плагин

## Инструмент 4: Faker
Для ускорения заполнения форм

## Инструмент 5: Lombok
Для разнообразия функций тестов

## Инструмент6: Gradle или Maven
Для создания автотестов

## Инструмент7: Allure
Для создания наглядных отчетов о результатах тестов
